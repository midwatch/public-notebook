Merge PDF Files
=======================================

**CLI:**

.. code-block:: console

    $ sudo apt-get update
    $ sudo apt-get install poppler-utils
    $ pdfunite source1.pdf source2.pdf merged_output.pdf

**GUI:**

.. code-block:: console

    $ sudo apt-get update
    $ sudo apt-get install pdfsam

    KDE -> All Applications -> Office -> PDFsam Basic
